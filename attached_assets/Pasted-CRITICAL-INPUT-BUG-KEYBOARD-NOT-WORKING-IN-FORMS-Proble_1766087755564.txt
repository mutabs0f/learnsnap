Affected Fields:
- Full Name input field
- Email input field  
- Password fields (likely also affected)

Current Behavior:
1. User clicks on input field
2. Field gets focus (cursor appears)
3. User types on keyboard
4. NO text appears in the field
5. Field remains empty

Expected Behavior:
Text should appear in the field as user types
