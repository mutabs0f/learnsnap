# LearnSnap - Development Prompt for Replit

## Project Overview
Build a B2C SaaS educational app called **LearnSnap** that transforms textbook pages into interactive learning experiences for elementary students (ages 6-12) in Saudi Arabia.

---

## Core Concept
**Parent Flow:**
1. Parent photographs 5-15 textbook pages
2. Uploads to LearnSnap
3. AI processes and generates interactive content (30-60 seconds)
4. Child completes 3 learning stages
5. Parent receives detailed progress report

**Child Flow:**
1. Learn: Simple explanations with visuals
2. Practice: 5 questions with instant feedback
3. Test: 10 questions measuring understanding
4. Results automatically sent to parent

## AI Processing Flow

```typescript
// Simplified 3-AI verification system

async function generateChapter(photos: File[]) {
  // Step 1: Generate with Gemini Flash (fast & cheap)
  const geminiContent = await generateWithGemini(photos);
  
  // Step 2: Verify with GPT-4o mini
  const verification = await verifyWithGPT(geminiContent);
  
  if (verification.status === "PASS") {
    return geminiContent; // âœ… Good to go!
  }
  
  // Step 3: Fix with Claude if needed
  const claudeFixed = await fixWithClaude(geminiContent, verification.issues);
  
  // Step 4: Re-verify
  const recheck = await verifyWithGPT(claudeFixed);
  
  if (recheck.status === "PASS") {
    return claudeFixed;
  }
  
  throw new Error("Content failed verification");
}
```
